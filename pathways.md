# Data Science Foundations - R

This module provides a practical, hands-on introduction to modern data science workflows using the R programming language and its Tidyverse ecosystem.

| Competency | Tools and Packages |
| :--- | :--- |
| Data Manipulation | dplyr, tidyr, pipe operator ($\%>\%$) |
| Data Visualization | ggplot2 |
| Statistical Analysis | t-tests, ANOVA, ANCOVA, DESeq2 |
| Cloud & AI Integration | Google Cloud Platform (Vertex AI), JupyterLab, Gemini |

# Introduction to Data Science for Biology

This module demystifies fundamental machine learning concepts by teaching the critical lesson of model decay and the necessity of retraining.

| Competency | Tools and Packages |
| :--- | :--- |
| Data Handling | Pandas, NumPy |
| Model Building | scikit-learn (sklearn) |
| ML Model | DecisionTreeRegressor |
| Model Maintenance | Model retraining to adapt to changing conditions |

| Competency | Tools and Packages |
